# Heading
  - Starts with `#` for h1, `##` for h2 and `###` for h3

@key: value
  - Metadata key-value pairs like:
    @title: My Blog
    @author: Bilal

Paragraph
  - Just write normal lines of text
  - Blank lines for separate paragraphs

* Bullet List
  - Lines starting with `*` are list items

1. Numbered List
  - Lines starting with `1.`, `2.`, etc. are ordered lists

--image: path
  - Insert an image
  - Example: --image: banner.png

--link: Text => URL
  - Insert a hyperlink
  - Example: --link: GitHub => https://github.com

// Comment
  - Any line starting with `//` is ignored
